📋 算法说明

本项目实现 vs 生产环境升级方向

✅ 本项目实现

  • 算法：DES-CBC（对称加密，分组密码）
  • 密钥：8 字节 b'FTP2025!'
  • IV：8 字节，由 md5(KEY) 派生
  • 填充：PKCS#7（块大小 = 8 字节）
  • 密码哈希：SHA-256（单向不可逆）
  • 权限模型：BLP 四级（公开/一般/高级/特殊）

🚀 生产环境升级方向

  • DES → AES-256-GCM（更强加密 + 认证）
  • 静态密钥 → RSA / Diffie-Hellman 密钥交换
  • SHA-256 → bcrypt / Argon2（抗暴力破解）
  • HTTP → TLS 1.3 整链路加密
  • FTP → SFTP / FTPS（标准协议）
  • 本地 JSON → 数据库 + RBAC（生产级权限）
